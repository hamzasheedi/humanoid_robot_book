# Create Docusaurus Structure Skill

## Overview

This skill automates the creation of a Docusaurus project structure for the Physical AI & Humanoid Robotics Textbook. It ensures all essential files and folders are generated in a ready-to-use state for documentation, making the project immediately deployable to GitHub Pages or any other hosting platform.

## Input

```json
{
  "projectName": "Name of the Docusaurus project"
}
```

## Output

```json
{
  "message": "Success message confirming project structure creation",
  "projectPath": "Path to the root of the newly created project"
}
```

## Resources

This skill includes resource directories that demonstrate how to organize different types of bundled resources:

### scripts/
Executable code (Python/Bash/etc.) that can be run directly to perform Docusaurus structure creation operations.

### references/
Documentation and reference material for Docusaurus configuration, theming, and best practices.

### assets/
Template files for Docusaurus configuration, sidebar, and package.json to be used in the generated project.

## Functionality

### Create Root Project Folder
- Creates a root folder with the given projectName
- Ensures the folder is created in the current working directory
- Checks for existing folders and prevents overwriting without warning

### Create docs/ Folder
- Creates a docs/ folder for Markdown chapters and other content
- Pre-populates with placeholder files if needed
- Ensures folder hierarchy is compatible with Docusaurus auto-generated sidebar

### Create sidebars.js
- Creates a basic sidebars.js file for Docusaurus
- Sets up tutorialSidebar using auto-generated structure
- Example:
```javascript
module.exports = {
  tutorialSidebar: [{ type: 'autogenerated', dirName: '.' }],
};
```

### Create docusaurus.config.js
- Creates a comprehensive configuration file for Docusaurus
- Sets project title and baseUrl using projectName
- Configures organizationName and projectName for GitHub Pages deployment
- Sets up presets for classic documentation with sidebar integration
- Includes navbar, footer, and theme configurations specific to the textbook

### Create Static Assets Folder
- Creates static/ folder for images, CSS, or other assets
- Ensures relative paths are compatible with Markdown references

### Create Package.json
- Creates a basic package.json file with Docusaurus dependencies
- Includes scripts for development, building, and deployment

### Validation
- Checks that all files are created properly
- Validates syntax of configuration files
- Returns projectPath and success message when complete
