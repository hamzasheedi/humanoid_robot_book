# MODULE DIRECTORY

This directory contains all modules for the AI Robotics Textbook:

## [ROS 2 (Robot Operating System 2)](/docs/modules/ros2/)
- Complete module for ROS 2 fundamentals and implementation
- Nodes, Topics, Services and their applications
- rclpy Python bridges for AI integration

## [DIGITAL-TWIN](/docs/modules/digital-twin/)
- Complete module for creating digital twins
- Gazebo physics simulation environments
- Unity visual rendering and interaction

## [NVIDIA-ISAAC](/docs/modules/nvidia-isaac/)
- Complete module for AI perception and navigation
- Isaac Sim setup and perception
- Nav2 path planning and VSLAM integration

## [VLA (Vision-Language-Action)](/docs/modules/vla/)
- Complete module for Vision-Language-Action systems
- Voice command processing with Whisper
- Cognitive planning with LLMs for robot control

## [CAPSTONE](/docs/modules/capstone/)
- Complete capstone project module
- Integration of all previous modules
- AI-powered humanoid robot implementation
