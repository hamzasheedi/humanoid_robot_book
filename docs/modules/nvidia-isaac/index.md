# NVIDIA-ISAAC

## Module Overview

AI-powered perception and navigation using NVIDIA's robotics platform. This module covers computer vision, machine learning for robotics, and autonomous navigation systems.

### Focus and Theme: *Intelligent perception and navigation for autonomous robots*

### Goal: *Students implement AI systems that understand and navigate the physical world*

## CONTENT

### [NVIDIA Isaac Introduction](./introduction.md)
Introduction to NVIDIA Isaac and its role in AI-powered robotics.

### [NVIDIA Isaac Perception Guide](./perception.md)
Using Isaac Sim for AI perception and computer vision tasks.

### [NVIDIA Isaac Navigation Guide](./navigation.md)
Implementing navigation and path planning with Isaac and Nav2.

### [Isaac Reinforcement Learning Modules](./reinforcement-learning.md)
Using Isaac for reinforcement learning applications.

### [NVIDIA Isaac Exercises](./exercises/)
Practice exercises for Isaac-based perception and navigation.

## LEARNING OBJECTIVES

- Implement AI perception systems using Isaac
- Use VSLAM for robot localization and mapping
- Create navigation systems with obstacle avoidance
- Apply reinforcement learning for robotic tasks
- Integrate perception with action systems
- Optimize AI models for deployment on robotic platforms

## PREREQUISITES

- Module 1 (ROS 2 Fundamentals) completed
- Module 2 (Digital Twin) completed
- Basic understanding of machine learning concepts

## WEEKLY BREAKDOWN

### Week 10: Isaac Navigation and AI
- Navigation with Isaac and Nav2

## ASSESSMENTS

- Perception pipeline implementation
- Navigation system performance validation
- VSLAM accuracy testing
- Reinforcement learning task completion

## HARDWARE SETUP

### Digital Twin Workstation (Required)
- NVIDIA GPU (RTX 4070 Ti minimum, RTX 4080+ recommended)
- CUDA 11.8+ compatible driver
- Isaac Sim license and setup

### Physical AI Edge Kit
- Jetson Orin AGX or equivalent GPU capability
- Isaac ROS packages installed
- Compatible sensors for perception tasks

### Alternative Cloud Setup
- AWS G5/G6 or Azure NVv4 instances with GPU
- Isaac Sim cloud deployment
- Local Jetson for final deployment validation
